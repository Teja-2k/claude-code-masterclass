### Post 3: r/learnprogramming

**Title:** How I use AI to code 10x faster — practical techniques, not hype

**Body:**

I see a lot of posts asking whether AI will replace programmers. I think that's the wrong question. The right question is: how do you use AI tools to become dramatically more productive?

I've been using Claude Code daily for months, and I want to share practical techniques that actually work. Not hype. Not "just ask ChatGPT to write your code." Real workflow changes.

**First, understand what Claude Code actually is.**

It's not a chatbot. It runs in your terminal. It can read your files, write new ones, run shell commands, and maintain context across your entire project. Think of it less like "AI autocomplete" and more like "a junior developer pair-programming with you who never gets tired."

**Technique 1: Write a CLAUDE.md file for every project.**

This is a markdown file at your project root that tells Claude Code about your project. Your tech stack, file structure, naming conventions, common patterns. Claude reads this automatically. It's the single biggest thing you can do to improve output quality.

**Technique 2: Break complex tasks into phases.**

Don't say "build me a full-stack app." Instead:
- "Set up the Next.js project with TypeScript and Tailwind"
- "Create the database schema for users and posts"
- "Build the API routes for CRUD operations"
- "Create the frontend components"

Each phase builds on the last. Claude handles each step better than trying to do everything at once.

**Technique 3: Let Claude run commands, don't just read the output.**

Claude Code can execute shell commands. Let it run your tests. Let it start the dev server and check for errors. Let it install dependencies. The more autonomy you give it (within reason), the faster you move.

**Technique 4: Use it for learning, not just building.**

When Claude writes code, ask it to explain the decisions. "Why did you use this pattern?" "What are the tradeoffs?" You'll learn faster than reading docs because the explanations are in the context of YOUR code.

**Technique 5: Review everything.**

AI makes you faster, not infallible. Read what it writes. Understand why. This is how you stay a developer who uses AI, rather than becoming someone who can't code without it.

**Going deeper:**

There's a lot more to Claude Code — MCP servers that connect it to databases and APIs, headless mode for automation, hooks for automated testing, multi-agent patterns. The more advanced stuff gets genuinely powerful.
